# About D-money project
Dmoney is a transactional system where users can send money, make payments, and conduct transactions

# Technology used:
- postman
- newman
- nodejs

# How to run
- Clone this project
- PUT .env file 
- Give the following command: ``` npm i ```
- ``` npm test ```
